# Run the SmartStorey app

**Important:** Run the dev server from THIS folder (the one inside your Cursor workspace).

1. Open Terminal in Cursor
2. Make sure you're in the right folder:
   ```bash
   pwd
   ```
   You should see: `.../Documents/SFMBA_Study/Cursor/smartstorey-app`

3. If you're in the Cursor folder, run:
   ```bash
   cd smartstorey-app
   ```

4. Start the server:
   ```bash
   npm run dev
   ```

5. Open in browser: **http://localhost:3000** (or 3001 if 3000 is in use)
